# 🎲 D&D Campaign

A modern web application built to document and explore a live **Dungeons & Dragons campaign**.

This project serves as a digital campaign archive where players and the game master can explore:

- 📖 Session logs and campaign events
- 🧙 Character sheets and stats
- 🖼 Campaign artwork and gallery
- 🌍 World lore, factions, locations, and artifacts
- 📝 Administrative tools for updating campaign content

---

## 🚀 Tech Stack

### Frontend
- **Next.js**
- **TypeScript**
- **App Router**
- **Tailwind CSS**
- **shadcn/ui**

### Backend / Data
- **Prisma ORM**
- **MySQL**

### Infrastructure
- **Azure Static Web Apps**
- **Azure Database for MySQL**
